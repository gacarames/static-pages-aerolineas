Font Installation

For step-by-step instructions on installing fonts on your computer, go to: http://www.typetrust.com/info/installation.php

Some Important Considerations When Handling Your Fonts:

  • Never rename any font files.
  • Do not install more than one format of any given font: Opentype, TrueType or PostScript Type 1.
  • Macintosh operating systems prior to Mac OS X require Adobe Type Manager [ATM] to make use of Opentype and PostScript Type 1 fonts.
  • Not all Opentype fonts are programmed with advanced typesetting features. Likewise, not every design application supports advanced Opentype features.

For OS and application specific installation instructions, please refer to the documentation supplied with the application.
